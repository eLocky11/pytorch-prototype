# PyTorch Formula Prototype

## Описание проекта

Это учебный прототип на **PyTorch**, созданный для быстрого знакомства с базовым циклом обучения нейросети.

Цель проекта — обучить простую модель находить зависимость между входным числом `x` и результатом `y` по формуле:

```text
y = 3x + 2
```

Проект не предназначен для реального применения. Его задача — показать минимальный рабочий пример машинного обучения:

1. подготовка данных;
2. создание модели;
3. расчёт ошибки;
4. обратное распространение ошибки;
5. обновление параметров модели;
6. проверка результата.

---

## Текущий статус

На данный момент проект состоит из одного основного скрипта:

```text
train.py
```

В нём находятся:

## Что изучается в этом прототипе

Проект помогает понять базовые элементы PyTorch:

### Tensor

Данные хранятся в виде тензоров:

```python
x_train = torch.tensor([[1.0], [2.0], [3.0]])
```

### Model

Модель создаётся через:

```python
model = nn.Linear(1, 1)
```

Она пытается выучить зависимость вида:

```text
y = wx + b
```

Где:

- `w` — вес;
- `b` — смещение;
- оба параметра подбираются автоматически во время обучения.

### Loss Function

Для оценки ошибки используется:

```python
loss_fn = nn.MSELoss()
```

Она показывает, насколько предсказания модели отличаются от правильных ответов.

### Optimizer

Для обновления параметров используется:

```python
optimizer = torch.optim.SGD(model.parameters(), lr=0.01)
```

### Training Loop

Основной цикл обучения:

```python
y_pred = model(x_train)
loss = loss_fn(y_pred, y_train)

optimizer.zero_grad()
loss.backward()
optimizer.step()
```

## Этап 6. Переход к нелинейной зависимости

После линейной формулы можно попробовать более сложную:

```text
y = x² + 2x + 1
```

Для такой задачи одной модели `nn.Linear(1, 1)` уже будет недостаточно.

Можно попробовать модель:

```python
model = nn.Sequential(
    nn.Linear(1, 16),
    nn.ReLU(),
    nn.Linear(16, 16),
    nn.ReLU(),
    nn.Linear(16, 1)
)
```

Это позволит понять, зачем нужны скрытые слои и функции активации.

# Цель проекта

Главная цель проекта — не создать сложную нейросеть, а понять фундаментальный принцип PyTorch:

```text
данные → модель → предсказание → ошибка → backward → optimizer → улучшение модели
```

После понимания этого цикла будет проще переходить к более сложным задачам:

- классификация;
- изображения;
- игровые данные;
- рекомендательные системы;
- собственные датасеты.
